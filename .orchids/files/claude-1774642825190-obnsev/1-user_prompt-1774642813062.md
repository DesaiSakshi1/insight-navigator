row called "🧠 AI-Powered Insights" with a glowing blue 
border and dark background.

messages based on the actual data analysis results. 
Generate these insights using rule-based logic:

Skewness insights:
- If any numeric column has skewness > 2: show "⚠️ 
[column] is highly right-skewed (skewness: X) — 
recommend log transformation before ML modeling"
- If skewness is between 0.5 and 2: show "📊 [column] 
shows moderate skewness — consider normalization"
- If skewness near 0: show "✅ [column] is normally 
distributed — ML ready"

Missing value insights:
- If missing % > 5: show "🔴 [column] has X% missing 
values — high risk, consider dropping or imputing"
- If missing % between 1-5: show "🟡 [column] has X% 
missing — recommend median/mode imputation"
- If 0 missing: show "✅ No missing values detected — 
dataset is clean"

Outlier insights:
- If outliers detected: show "⚠️ X outliers found in 
[column] — investigate before predictive modeling"

Correlation insights:
- If strong correlation > 0.7: show "🔗 Strong 
correlation detected between [A] and [B] (r=X) — 
potential multicollinearity risk"
- If no strong correlation: show "✅ No multicollinearity 
detected — features are independent"

General insights:
- Show "📋 Dataset has X rows and Y columns — 
[Small/Medium/Large] sized dataset"
- If duplicate rows > 0: show "🔁 X duplicate rows 
found — remove before training any ML model"
- Show "🏆 Top performing category: [category with 
highest count]"
- Show "📍 Most common region: [region with highest 
orders]"

Each insight must be on its own row with:
- Colored emoji icon on left
- Bold insight text
- Light grey subtext explaining why it matters
- Subtle divider between each insight

At the bottom of the card show a green badge: 
"✓ Analysis Complete — X insights generated"

Make all insights dynamic based on real analyzed data.

in one by one with 0.1s delay between each.
